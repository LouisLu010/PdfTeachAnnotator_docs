# 快速开始

## 安装

### 下载

从 [GitHub Releases](https://github.com/LouisLu010/PdfTeachAnnotator/releases) 页面下载最新版本：

- **发布包**: `PdfTeachAnnotator-Release.zip`

### 系统要求

- **操作系统**: Windows 10/11 (x64)
- **运行环境**: .NET 8.0 或更高版本
- **内存**: 建议 4GB 以上
- **磁盘空间**: 约 207 MB（包含完整运行时和 OCR 语言包）

### 安装步骤

1. 下载 `PdfTeachAnnotator-Release.zip`
2. 解压到任意目录
3. 双击 `PdfTeachAnnotator.lnk` 快捷方式启动

::: tip 提示
首次运行可能需要等待几秒钟加载。所有依赖已包含在发布包中，无需额外安装。
:::

## 首次使用

### 打开 PDF 文件

有多种方式打开 PDF 文件：

1. **点击按钮**: 点击主页的"打开 PDF"按钮或工具栏的📂图标，在文件对话框中选择文件
2. **快捷键**: 按 `Ctrl+O` 打开文件对话框
3. **拖拽文件**: 直接将 PDF 文件拖拽到应用窗口
4. **最近文件**: 在主页点击最近访问的文件卡片快速打开

### 开始批注

1. 文件打开后，工具栏会自动显示
2. 点击画笔工具（✏️图标）
3. 选择颜色和粗细（点击对应的圆圈）
4. 在 PDF 页面上拖动鼠标绘制批注

### 保存批注

- **手动保存**: 点击工具栏的保存按钮（💾图标）或按 `Ctrl+S`
- **自动保存**: 在设置中启用"自动保存批注"，关闭文件时自动保存

::: tip 提示
批注数据保存为独立的 `.annotations` 文件，与 PDF 文件同目录，不会修改原始 PDF 文件。
:::

## 基本操作

### 工具栏

工具栏位于窗口顶部，包含以下工具：

| 工具 | 说明 | 操作方式 |
|------|------|--------|
| 菜单 | 打开主菜单（返回主页、设置等） | 点击 ≡ 图标 |
| 打开 | 打开 PDF 文件 | 点击 📂 图标 |
| 保存 | 保存批注 | 点击 💾 图标 |
| 撤销 | 撤销上一笔批注 | 点击 ↶ 图标 |
| 重做 | 重做刚才撤销的操作 | 点击 ↷ 图标 |
| 画笔 | 绘制批注 | 点击 ✏️ 图标 |
| 橡皮 | 擦除批注 | 点击 🧹 图标 |
| 清空 | 清空所有批注（需滑动确认） | 点击 🗑️ 图标 |
| 放大 | 增加缩放级别 | 点击 🔍+ 图标 |
| 缩小 | 减少缩放级别 | 点击 🔍- 图标 |
| 重置 | 恢复到 100% 缩放 | 点击 🔄 图标 |

### 缩放视图

- **放大**: 点击放大按钮（🔍+）或按住 `Ctrl` 滚动鼠标滚轮向上
- **缩小**: 点击缩小按钮（🔍-）或按住 `Ctrl` 滚动鼠标滚轮向下
- **重置**: 点击重置按钮（🔄）恢复到 100% 缩放
- **缩放范围**: 支持 50% - 400% 的缩放级别
- **显示**: 工具栏实时显示当前缩放百分比

### 颜色选择

工具栏提供 8 种预设颜色：

- 红色 🔴
- 蓝色 🔵
- 绿色 🟢
- 黄色 🟡
- 橙色 🟠
- 紫色 🟣
- 黑色 ⚫
- 白色 ⚪

点击颜色圆圈即可切换画笔颜色，当前选中的颜色会有视觉高亮。

### 笔粗调整

工具栏提供多种笔粗选项，可调范围：

- **默认范围**: 1-20 像素
- **快速选择**: 点击工具栏圆点切换常用粗细
- **自定义**: 在设置中配置默认画笔粗细

常用粗细推荐：
- 细（1-3px）：适合精细标注
- 中（4-5px）：适合一般批注
- 粗（6-8px）：适合强调重点
- 特粗（10-12px）：适合演示或投影

### 橡皮擦大小

工具栏提供多种橡皮擦大小，可调范围：

- **默认范围**: 5-50 像素
- **快速选择**: 点击工具栏圆点切换常用大小
- **自定义**: 在设置中配置默认橡皮擦大小

常用大小推荐：
- 小（10px）：精确擦除
- 中（20px）：一般擦除
- 大（30px）：快速擦除
- 特大（40-50px）：大面积擦除
