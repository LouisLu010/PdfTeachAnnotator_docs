# 快速开始

## 安装

### 下载

从 [GitHub Releases](https://github.com/LouisLu010/PdfTeachAnnotator/releases) 页面下载最新版本：

- **Windows 安装包**: `PdfTeachAnnotator-Setup.exe`
- **便携版**: `PdfTeachAnnotator-Portable.zip`

### 系统要求

- **操作系统**: Windows 10 或更高版本
- **运行环境**: .NET 8.0 Runtime（安装包会自动检测并提示安装）
- **磁盘空间**: 至少 100 MB 可用空间

### 安装步骤

1. 双击下载的 `PdfTeachAnnotator-Setup.exe`
2. 按照安装向导提示完成安装
3. 安装完成后，从开始菜单或桌面快捷方式启动应用

## 首次使用

### 打开 PDF 文件

有三种方式打开 PDF 文件：

1. **点击按钮**: 点击主页的"打开 PDF"按钮，在文件对话框中选择文件
2. **拖拽文件**: 直接将 PDF 文件拖拽到应用窗口
3. **快捷键**: 按 `Ctrl+O` 打开文件对话框

### 开始批注

1. 文件打开后，工具栏会自动显示
2. 点击画笔工具（或按快捷键）
3. 选择颜色和粗细
4. 在 PDF 页面上绘制批注

### 保存批注

- **手动保存**: 点击保存按钮或按 `Ctrl+S`
- **自动保存**: 在设置中启用"自动保存批注"，关闭文件时自动保存

## 基本操作

### 工具栏

工具栏位于窗口顶部，包含以下工具：

| 工具 | 说明 | 快捷键 |
|------|------|--------|
| 菜单 | 打开主菜单 | - |
| 打开 | 打开 PDF 文件 | `Ctrl+O` |
| 保存 | 保存批注 | `Ctrl+S` |
| 画笔 | 绘制批注 | - |
| 橡皮 | 擦除批注 | - |
| 清空 | 清空所有批注 | - |

### 缩放视图

- **放大**: 点击放大按钮或按 `Ctrl+鼠标滚轮向上`
- **缩小**: 点击缩小按钮或按 `Ctrl+鼠标滚轮向下`
- **重置**: 点击重置按钮恢复到 100% 缩放

### 颜色选择

工具栏提供 8 种预设颜色：

- 红色 🔴
- 蓝色 🔵
- 绿色 🟢
- 黄色 🟡
- 橙色 🟠
- 紫色 🟣
- 粉色 🩷
- 黑色 ⚫

点击颜色圆圈即可切换画笔颜色。

### 笔粗调整

工具栏提供 4 种笔粗选项：

- 细 (2px)
- 中 (4px)
- 粗 (6px)
- 特粗 (8px)

点击对应的圆点即可切换笔粗。
