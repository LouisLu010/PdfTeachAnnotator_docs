# 功能介绍

## 批注工具

### 画笔工具

画笔是主要的批注工具，用于在 PDF 上绘制标记。

**特点：**
- 8 种预设颜色可选
- 4 种笔粗可调（2px、4px、6px、8px）
- 支持平滑绘图（可在设置中启用）
- 自然的手写体验

**使用方法：**
1. 点击工具栏中的画笔图标
2. 选择颜色和笔粗
3. 在 PDF 页面上按住鼠标左键拖动绘制

### 橡皮擦工具

橡皮擦用于擦除不需要的批注。

**特点：**
- 3 种橡皮大小可选（小、中、大）
- 精确擦除单个笔画
- 不会影响 PDF 原始内容

**使用方法：**
1. 点击工具栏中的橡皮图标
2. 选择橡皮大小
3. 在要擦除的批注上拖动鼠标

### 清空所有批注

快速清除当前 PDF 的所有批注。

**使用方法：**
1. 点击工具栏中的清空图标
2. 滑动滑块到右侧确认清空
3. 或点击"取消"按钮放弃操作

::: warning 注意
清空操作不可撤销，请谨慎使用。建议在清空前先保存当前批注。
:::

## 文件管理

### 打开 PDF

支持多种方式打开 PDF 文件：

**方式一：文件对话框**
- 点击"打开 PDF"按钮
- 或按 `Ctrl+O` 快捷键
- 在对话框中选择 PDF 文件

**方式二：拖拽文件**
- 直接将 PDF 文件拖拽到应用窗口
- 支持从文件资源管理器拖拽

**方式三：最近文件**
- 在主页查看最近访问的文件列表
- 点击文件卡片快速打开

### 保存批注

批注数据保存为独立的 `.annotations` 文件，与原 PDF 文件同名。

**手动保存：**
- 点击保存按钮
- 或按 `Ctrl+S` 快捷键

**自动保存：**
- 在设置中启用"自动保存批注"
- 关闭文件时自动保存

::: tip 提示
批注文件与 PDF 文件分离存储，不会修改原始 PDF 文件。
:::

### 最近文件

应用会自动记录最近访问的 PDF 文件。

**功能：**
- 显示文件名、页数、最后访问时间
- 显示是否包含批注
- 最多记录 20 个文件（可在设置中调整）
- 支持清空所有记录

## 视图控制

### 缩放功能

灵活调整 PDF 显示大小。

**缩放范围：** 50% - 400%

**缩放方式：**
- 点击放大/缩小按钮
- 按 `Ctrl+鼠标滚轮`
- 点击重置按钮恢复 100%

**显示：**
- 工具栏实时显示当前缩放百分比
- 缩放时保持页面居中

### 滚动浏览

**垂直滚动：**
- 鼠标滚轮上下滚动
- 拖动右侧滚动条

**水平滚动：**
- 按住 `Shift+鼠标滚轮`
- 拖动底部滚动条

## 界面布局

### 主页视图

应用启动时显示的欢迎页面。

**内容：**
- 快速操作区：打开 PDF 按钮
- 最近文件列表
- 空状态提示

### PDF 视图

打开 PDF 后的批注工作区。

**布局：**
- 顶部：工具栏
- 中间：PDF 页面（支持滚动）
- 页面：卡片式设计，带阴影效果

### 设置视图

个性化配置应用行为。

**分类：**
- 常规设置
- 绘图设置
- 视图设置
- 最近文件设置
- 外观设置

## 主题支持

应用支持浅色和深色主题。

**主题选项：**
- 浅色主题
- 深色主题
- 跟随系统（自动切换）

**设置方法：**
1. 打开设置页面
2. 在"外观"部分选择主题
3. 点击"保存设置"

::: tip 提示
主题切换会立即生效，无需重启应用。
:::

## 窗口管理

### 窗口状态

- 支持最大化/最小化/还原
- 支持拖动调整窗口大小
- 批注在窗口状态变化时自动保留

### 标题栏

采用 Fluent Design 风格的自定义标题栏。

**特点：**
- 标题栏与内容区域融合
- 支持拖动移动窗口
- 标准的窗口控制按钮

## 性能优化

### 渲染优化

- 按需渲染 PDF 页面
- 窗口大小变化时自动调整渲染质量
- 批注数据独立管理，不影响 PDF 渲染

### 内存管理

- 及时释放不需要的资源
- 关闭文件时清理内存
- 支持大型 PDF 文件

## 数据安全

### 批注存储

- 批注数据保存为二进制格式
- 与 PDF 文件分离存储
- 不修改原始 PDF 文件

### 自动保存

- 可选的自动保存功能
- 关闭应用时提示保存未保存的批注
- 防止意外丢失批注数据
