# 功能介绍

## OCR 文字识别

### 概述

应用内置 OCR（光学字符识别）功能，可以识别 PDF 文档中的文字内容。

**特点：**
- 基于 Tesseract 5.2.0 OCR 引擎
- 支持中文简体和英文混合识别
- 整篇 PDF 文档识别
- 实时显示识别进度
- 识别结果一键复制
- 高精度识别（2000px 渲染质量）

### 使用方法

1. 打开 PDF 文件
2. 点击侧边栏的"🧰工具箱"图标
3. 在 OCR 文字识别卡片中点击"开始识别"
4. 等待识别完成（显示进度：第 X/Y 页）
5. 识别完成后，文本显示在结果框中
6. 点击"复制结果"可复制识别文本到剪贴板

### 识别说明

**适用场景：**
- 扫描版 PDF 文档
- 图片格式的文字内容
- 中英文混合文本
- 需要提取文字进行编辑或搜索

**注意事项：**
- 识别速度取决于 PDF 页数和复杂度
- 图片质量较低或手写文字可能影响准确度
- 识别过程中可以进行其他操作（如批注）
- 大型文档识别可能需要较长时间

::: tip 提示
OCR 识别基于页面渲染图像，识别质量受原始 PDF 清晰度影响。建议对清晰的扫描件或图片进行识别。
:::

## 批注工具

### 画笔工具

画笔是主要的批注工具，用于在 PDF 上绘制标记。

**特点：**
- 8 种预设颜色可选（红、蓝、绿、黄、橙、紫、黑、白）
- 可调笔粗范围：1-20 像素（在设置中配置）
- 支持平滑绘图（可在设置中启用）
- 自然的手写体验
- 笔画颜色预览（画笔图标显示当前选中颜色）

**使用方法：**
1. 点击工具栏中的画笔图标（✏️）
2. 选择颜色（点击颜色圆圈）
3. 选择笔粗（点击圆点或在设置中配置）
4. 在 PDF 页面上按住鼠标左键拖动绘制

### 橡皮擦工具

橡皮擦用于擦除不需要的批注。

**特点：**
- 可调橡皮大小范围：5-50 像素（在设置中配置）
- 精确擦除单个笔画（基于笔画级擦除，不是像素级）
- 不会影响 PDF 原始内容
- 实时视觉反馈

**使用方法：**
1. 点击工具栏中的橡皮图标（🧹）
2. 选择橡皮大小（点击圆点或在设置中配置）
3. 在要擦除的批注上拖动鼠标（接触到笔画即可擦除整条笔画）

::: tip 提示
橡皮擦采用笔画级擦除，只要鼠标经过的区域接触到批注笔画，整条笔画就会被擦除。
:::

### 清空所有批注

快速清除当前 PDF 的所有批注，带滑动确认机制防止误操作。

**使用方法：**
1. 点击工具栏中的清空图标（🗑️）
2. 出现滑动确认条
3. 向右滑动滑块到底
4. 显示绿色勾号后自动清空所有批注
5. 点击"取消"按钮可放弃操作

::: warning 注意
清空操作不可撤销。建议在清空前先保存当前批注，或确认不再需要这些批注。
:::

### 撤销/重做

支持撤销和重做批注操作，提供容错机制。

**特点：**
- 支持多次撤销和重做
- 保留完整的操作历史
- 工具栏按钮实时显示可用状态
- 关闭文件后历史记录清空

**使用方法：**
- **撤销**: 点击工具栏的撤销按钮（↶）
- **重做**: 点击工具栏的重做按钮（↷）

::: tip 提示
撤销/重做功能基于命令模式实现，每次绘制的笔画和擦除操作都会记录，可以精确恢复。
:::

## 文件管理

### 打开 PDF

支持多种方式打开 PDF 文件：

**方式一：文件对话框**
- 点击主页的"打开 PDF"按钮
- 或点击工具栏的📂图标
- 在对话框中选择 PDF 文件

**方式二：拖拽文件**
- 直接将 PDF 文件拖拽到应用窗口
- 支持从文件资源管理器拖拽
- 可以拖拽到窗口任意位置

**方式三：最近文件**
- 在主页查看最近访问的文件列表
- 点击文件卡片快速打开
- 显示文件信息：名称、页数、批注状态、最后访问时间

### 保存批注

批注数据保存为独立的 `.annotations` 文件，与原 PDF 文件同名同目录。

**手动保存：**
- 点击工具栏的保存按钮（💾图标）
- 批注立即保存到磁盘

**自动保存：**
- 在设置中启用"自动保存批注"
- 关闭文件或退出应用时自动保存
- 无需手动操作

**批注文件格式：**
- 文件名：`{PDF文件名}.annotations`
- 位置：与 PDF 文件同目录
- 格式：二进制序列化格式（基于 InkCanvas 的 ISF 格式）

::: tip 提示
批注文件与 PDF 文件分离存储，不会修改原始 PDF 文件。迁移或备份时，需要同时复制 PDF 文件和对应的 `.annotations` 文件。
:::

### 最近文件

应用会自动记录最近访问的 PDF 文件，方便快速访问。

**功能：**
- 显示文件名、完整路径
- 显示页数
- 显示是否包含批注（图标标识）
- 显示最后访问时间（相对时间，如"2小时前"）
- 最多记录 5-20 个文件（可在设置中调整，默认 10 个）
- 支持一键清空所有记录

**操作：**
- 点击文件卡片：打开对应的 PDF 文件
- 清空记录：设置 → 最近文件 → 清空最近访问

::: tip 提示
清空最近访问记录不会删除 PDF 文件或批注文件，只是清除访问历史记录。
:::

## 视图控制

### 缩放功能

灵活调整 PDF 显示大小。

**缩放范围：** 50% - 400%

**缩放方式：**
- 点击放大/缩小按钮
- 按 `Ctrl+鼠标滚轮`
- 点击重置按钮恢复 100%

**显示：**
- 工具栏实时显示当前缩放百分比
- 缩放时保持页面居中

### 滚动浏览

**垂直滚动：**
- 鼠标滚轮上下滚动
- 拖动右侧滚动条

**水平滚动：**
- 按住 `Shift+鼠标滚轮`
- 拖动底部滚动条

## 界面布局

### 主页视图

应用启动时显示的欢迎页面。

**内容：**
- 快速操作区：打开 PDF 按钮
- 最近文件列表
- 空状态提示

### PDF 视图

打开 PDF 后的批注工作区。

**布局：**
- 顶部：工具栏
- 中间：PDF 页面（支持滚动）
- 页面：卡片式设计，带阴影效果

### 设置视图

个性化配置应用行为。

**分类：**
- 常规设置
- 绘图设置
- 视图设置
- 最近文件设置
- 外观设置

## 主题支持

应用支持浅色和深色主题。

**主题选项：**
- 浅色主题
- 深色主题
- 跟随系统（自动切换）

**设置方法：**
1. 打开设置页面
2. 在"外观"部分选择主题
3. 点击"保存设置"

::: tip 提示
主题切换会立即生效，无需重启应用。
:::

## 窗口管理

### 窗口状态

- 支持最大化/最小化/还原
- 支持拖动调整窗口大小
- 批注在窗口状态变化时自动保留

### 标题栏

采用 Fluent Design 风格的自定义标题栏。

**特点：**
- 标题栏与内容区域融合
- 支持拖动移动窗口
- 标准的窗口控制按钮

## 性能优化

### 渲染优化

- 按需渲染 PDF 页面
- 窗口大小变化时自动调整渲染质量
- 批注数据独立管理，不影响 PDF 渲染

### 内存管理

- 及时释放不需要的资源
- 关闭文件时清理内存
- 支持大型 PDF 文件

## 数据安全

### 批注存储

- 批注数据保存为二进制格式
- 与 PDF 文件分离存储
- 不修改原始 PDF 文件

### 自动保存

- 可选的自动保存功能
- 关闭应用时提示保存未保存的批注
- 防止意外丢失批注数据
