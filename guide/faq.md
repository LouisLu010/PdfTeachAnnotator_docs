# 常见问题

## 安装问题

### 提示缺少 .NET Runtime

**问题：** 启动应用时提示缺少 .NET 8.0 Runtime。

**解决方案：**
1. 访问 [.NET 下载页面](https://dotnet.microsoft.com/download/dotnet/8.0)
2. 下载并安装 .NET 8.0 Desktop Runtime
3. 重启应用

### 安装失败

**问题：** 安装程序无法完成安装。

**解决方案：**
1. 以管理员身份运行安装程序
2. 检查是否有足够的磁盘空间（至少 100 MB）
3. 关闭杀毒软件后重试
4. 检查 Windows 更新是否完成

## 使用问题

### 无法打开 PDF 文件

**问题：** 点击"打开 PDF"后没有反应或提示错误。

**可能原因：**
- PDF 文件损坏
- PDF 文件被其他程序占用
- 文件路径包含特殊字符

**解决方案：**
1. 尝试用其他 PDF 阅读器打开该文件，确认文件是否正常
2. 关闭其他正在使用该 PDF 的程序
3. 将文件移动到路径简单的文件夹（如桌面）
4. 尝试打开其他 PDF 文件，确认是否是应用问题

### 批注无法保存

**问题：** 点击保存按钮后批注没有保存。

**可能原因：**
- 没有写入权限
- 磁盘空间不足
- 文件路径过长

**解决方案：**
1. 确认 PDF 文件所在文件夹有写入权限
2. 检查磁盘空间是否充足
3. 将 PDF 文件移动到路径较短的位置
4. 以管理员身份运行应用

### 批注消失了

**问题：** 之前保存的批注打开后不见了。

**可能原因：**
- 批注文件被删除或移动
- 批注文件损坏
- PDF 文件被重命名或移动

**解决方案：**
1. 检查 PDF 文件同目录下是否有 `.annotations` 文件
2. 确认 PDF 文件名是否被修改
3. 如果 PDF 文件被移动，将批注文件也移动到同一目录
4. 如果批注文件损坏，可能无法恢复

::: tip 提示
批注文件与 PDF 文件同名，扩展名为 `.annotations`。例如：
- PDF 文件：`lecture.pdf`
- 批注文件：`lecture.annotations`
:::

### 窗口最大化后批注消失

**问题：** 切换窗口最大化/最小化时批注消失。

**解决方案：**
此问题已在最新版本中修复。请更新到最新版本。

### 绘制的线条不流畅

**问题：** 绘制的线条有锯齿或不平滑。

**解决方案：**
1. 在设置中启用"平滑绘图"选项
2. 尝试调整画笔粗细
3. 检查鼠标或触控板是否正常工作

### 橡皮擦无法擦除

**问题：** 使用橡皮擦工具无法擦除批注。

**可能原因：**
- 橡皮大小太小
- 没有准确点击到笔画

**解决方案：**
1. 增大橡皮大小
2. 确保鼠标点击在笔画上
3. 尝试多次擦除

## 性能问题

### 应用启动慢

**问题：** 应用启动需要很长时间。

**解决方案：**
1. 检查系统资源占用情况
2. 关闭不必要的后台程序
3. 重启计算机
4. 检查杀毒软件是否在扫描应用

### 打开大型 PDF 很慢

**问题：** 打开页数较多的 PDF 文件需要很长时间。

**解决方案：**
1. 这是正常现象，应用需要渲染所有页面
2. 等待渲染完成后即可正常使用
3. 考虑将大型 PDF 拆分为多个小文件

### 缩放时卡顿

**问题：** 调整缩放级别时界面卡顿。

**解决方案：**
1. 减少批注数量
2. 关闭其他占用资源的程序
3. 升级计算机硬件（特别是内存）

## 数据问题

### 如何备份批注

**问题：** 如何备份我的批注数据？

**解决方案：**
批注保存在 `.annotations` 文件中，与 PDF 文件同目录。备份方法：
1. 复制 PDF 文件和对应的 `.annotations` 文件
2. 将两个文件保存到备份位置
3. 恢复时将两个文件放回同一目录

### 如何迁移到新电脑

**问题：** 如何将批注数据迁移到新电脑？

**解决方案：**
1. 复制所有 PDF 文件和对应的 `.annotations` 文件
2. 在新电脑上安装应用
3. 将文件复制到新电脑
4. 打开 PDF 文件，批注会自动加载

### 如何导出批注

**问题：** 如何将批注导出为图片或将批注合并到 PDF？

**解决方案：**

当前版本（v1.0）不支持直接导出功能。临时解决方案：

1. **截图方式**：使用 Windows 截图工具（Win+Shift+S）截取带批注的页面
2. **打印方式**：使用打印功能将页面打印为 PDF（需要第三方虚拟打印机）
3. **第三方工具**：使用专业 PDF 编辑工具手动合并

::: tip 提示
我们计划在未来版本中添加以下导出功能：
- 导出为带批注的 PDF（将批注烧录到 PDF 页面）
- 导出单页为图片（PNG/JPG）
- 导出批注为透明图层
:::

### 撤销和重做功能如何使用

**问题：** 如何撤销错误的批注？

**解决方案：**

应用支持完整的撤销/重做功能：

- **撤销**：按 `Ctrl+Z` 或点击工具栏的撤销按钮（↶图标）
- **重做**：按 `Ctrl+Y` 或 `Ctrl+Shift+Z`，或点击工具栏的重做按钮（↷图标）
- **多次撤销**：可以连续按多次，回退到任意历史状态
- **历史记录**：关闭文件后历史记录清空

## 兼容性问题

### 支持哪些 PDF 版本

**问题：** 应用支持哪些版本的 PDF 文件？

**解答：**
应用支持大多数标准 PDF 文件（PDF 1.0 - 2.0）。不支持：
- 加密的 PDF 文件
- 需要密码的 PDF 文件
- 损坏的 PDF 文件

### 支持触摸屏吗

**问题：** 应用支持触摸屏设备吗？

**解答：**
支持。可以使用触摸屏进行绘制和操作。但触控笔体验更好。

### 支持触控笔吗

**问题：** 应用支持触控笔（如 Surface Pen）吗？

**解答：**
支持。应用完全支持 Windows Ink 触控笔，可以获得更自然的书写体验。

**触控笔功能：**
- 支持压感（如果触控笔支持）
- 支持倾斜角度（部分触控笔）
- 笔尖绘制，橡皮擦擦除（如果触控笔支持翻转擦除）
- 悬浮预览（取决于硬件支持）

### 为什么快捷键不工作

**问题：** 按 Ctrl+O、Ctrl+S 等快捷键没有反应。

**解答：**
应用已实现以下键盘快捷键：

- `Ctrl+O` - 打开 PDF 文件
- `Ctrl+S` - 保存批注
- `Ctrl+Z` - 撤销
- `Ctrl+Y` - 重做
- `Ctrl+Shift+Z` - 重做

**可能的问题：**
1. 焦点不在主窗口上：点击窗口任意位置获取焦点
2. 其他程序占用了快捷键：关闭可能冲突的程序
3. 输入法拦截：切换到英文输入状态

::: tip 提示
如果快捷键仍不工作，可以使用工具栏按钮或鼠标操作作为替代方案。
:::

### 如何更改默认设置

**问题：** 如何更改默认的画笔粗细、颜色等设置？

**解决方案：**
1. 点击工具栏菜单按钮（≡图标）
2. 选择"设置"
3. 在设置页面调整以下选项：
   - 默认画笔粗细（1-20px）
   - 默认橡皮擦大小（5-50px）
   - 默认画笔颜色（8种颜色可选）
   - 默认缩放级别（50%-200%）
   - 其他选项（自动保存、平滑绘图等）
4. 点击"保存设置"按钮

::: tip 提示
设置保存在 `%APPDATA%\PdfTeachAnnotator\settings.json`，如需恢复默认设置，可删除此文件。
:::

### 批注文件可以在其他设备上打开吗

**问题：** 我能在另一台电脑上打开相同的批注吗？

**解答：**
可以。批注数据保存在独立的 `.annotations` 文件中。

**迁移步骤：**
1. 复制 PDF 文件和对应的 `.annotations` 文件到新设备
2. 在新设备上安装相同版本的应用
3. 使用应用打开 PDF 文件
4. 批注会自动加载

::: warning 注意
必须同时复制 PDF 文件和 `.annotations` 文件，且保持它们在同一目录下。
:::

### 如何卸载

**问题：** 如何完全卸载应用？

**解决方案：**
1. 打开 Windows 设置 → 应用 → 已安装的应用
2. 找到"PDF 教学批注工具"
3. 点击"卸载"
4. 手动删除配置文件（可选）：
   - `%APPDATA%\PdfTeachAnnotator\`

::: warning 注意
卸载应用不会删除 PDF 文件和批注文件。
:::
