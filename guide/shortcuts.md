# 快捷键

## 文件操作

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl+O` | 打开 PDF | 打开文件对话框选择 PDF 文件 |
| `Ctrl+S` | 保存批注 | 保存当前 PDF 的批注数据 |

## 编辑操作

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl+Z` | 撤销 | 撤销上一笔批注或擦除操作 |
| `Ctrl+Y` | 重做 | 重做刚才撤销的批注操作 |
| `Ctrl+Shift+Z` | 重做 | 与 `Ctrl+Y` 相同 |

## 视图控制

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl+鼠标滚轮向上` | 放大 | 增加缩放级别（每次约 10%） |
| `Ctrl+鼠标滚轮向下` | 缩小 | 减少缩放级别（每次约 10%） |

## 鼠标操作

| 操作 | 功能 | 说明 |
|------|------|------|
| `鼠标滚轮` | 垂直滚动 | 上下滚动 PDF 页面 |
| `Shift+鼠标滚轮` | 水平滚动 | 左右滚动 PDF 页面（如果页面宽度超出视口） |
| 左键拖动 | 绘制/擦除 | 在画笔模式下绘制，在橡皮模式下擦除 |
| 拖拽文件 | 打开 PDF | 将 PDF 文件拖拽到窗口打开 |

## 工具栏操作

工具栏提供直观的点击式操作界面：

### 颜色选择

点击工具栏中的颜色圆圈快速切换画笔颜色：

- 🔴 红色
- 🔵 蓝色
- 🟢 绿色
- 🟡 黄色
- 🟠 橙色
- 🟣 紫色
- ⚫ 黑色
- ⚪ 白色

### 画笔粗细选择

点击工具栏中的圆点快速切换画笔粗细：

- 细：1-3px
- 中：4-5px
- 粗：6-8px
- 特粗：10-12px

（具体可用粗细在设置中配置，默认范围 1-20px）

### 橡皮擦大小选择

点击工具栏中的圆点快速切换橡皮擦大小：

- 小：10px
- 中：20px
- 大：30px
- 特大：40-50px

（具体可用大小在设置中配置，默认范围 5-50px）

## 常用操作流程

### 快速批注流程

1. 按 `Ctrl+O` 打开 PDF（或点击按钮/拖拽文件）
2. 点击画笔工具（工具栏左侧）
3. 选择颜色和粗细（点击对应圆圈）
4. 左键拖动绘制批注
5. 按 `Ctrl+S` 保存批注（或启用自动保存）

### 快速修改流程

1. 点击橡皮擦工具
2. 选择橡皮擦大小
3. 在错误批注上拖动擦除
4. 如需撤销，按 `Ctrl+Z`
5. 点击画笔工具继续批注
6. 按 `Ctrl+S` 保存批注

### 快速浏览流程

1. 按住 `Ctrl` 键，滚动鼠标滚轮调整缩放
2. 鼠标滚轮上下滚动浏览页面
3. 按住 `Shift` 键，滚动鼠标滚轮左右浏览（如页面宽度超出视口）

## 效率技巧

### 技巧 1：快捷键组合使用

- `Ctrl+O` 打开文件 → 批注 → `Ctrl+S` 保存
- 出错时立即按 `Ctrl+Z` 撤销，比使用橡皮擦更快
- `Ctrl+Y` 或 `Ctrl+Shift+Z` 恢复误撤销的操作

### 技巧 2：快速切换工具

- 画笔和橡皮擦工具点击一次即可切换
- 颜色和粗细选择也只需一次点击
- 所有工具都有视觉反馈（高亮显示当前选中项）

### 技巧 3：缩放技巧

- 按住 `Ctrl` 键，使用鼠标滚轮快速调整缩放
- 支持的缩放范围：50% - 400%
- 缩放时页面会自动重新渲染，保持清晰度

### 技巧 4：拖拽打开

- 直接从文件资源管理器拖拽 PDF 文件到窗口
- 比按 `Ctrl+O` 更快捷
- 支持拖拽到窗口任意位置

### 技巧 5：最近文件

- 在主页点击最近文件卡片快速打开
- 显示文件名、页数、批注状态和最后访问时间
- 比每次按 `Ctrl+O` 浏览文件夹更高效

### 技巧 6：自动保存

- 在设置中启用"自动保存批注"
- 关闭文件时自动保存批注，无需记住按 `Ctrl+S`
- 防止批注意外丢失

### 技巧 7：撤销/重做

- `Ctrl+Z` 快速撤销错误操作
- `Ctrl+Y` 快速恢复
- 支持多次撤销和重做
- 批注历史记录保持在内存中，关闭文件后清空

我们计划在未来版本中添加以下功能：

- `Ctrl+W` - 关闭当前文件
- `Ctrl+Q` - 退出应用

- `Ctrl+0` - 重置缩放
- `P` / `E` - 切换画笔/橡皮擦

- 预设快捷键方案（如 Adobe 风格）

### 辅助功能

- 屏幕阅读器支持
- 高对比度主题
- 更大的按钮和文字选项
- 完整的键盘导航
