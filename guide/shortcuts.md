# 快捷键

## 文件操作

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl+O` | 打开 PDF | 打开文件对话框选择 PDF 文件 |
| `Ctrl+S` | 保存批注 | 保存当前 PDF 的批注数据 |

## 视图控制

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl+鼠标滚轮向上` | 放大 | 增加缩放级别（每次 10%） |
| `Ctrl+鼠标滚轮向下` | 缩小 | 减少缩放级别（每次 10%） |
| `鼠标滚轮` | 垂直滚动 | 上下滚动 PDF 页面 |
| `Shift+鼠标滚轮` | 水平滚动 | 左右滚动 PDF 页面 |

## 鼠标操作

| 操作 | 功能 | 说明 |
|------|------|------|
| 左键拖动 | 绘制/擦除 | 在画笔模式下绘制，在橡皮模式下擦除 |
| 拖拽文件 | 打开 PDF | 将 PDF 文件拖拽到窗口打开 |

## 工具栏快捷操作

虽然工具栏按钮没有键盘快捷键，但可以通过以下方式快速操作：

### 颜色切换

点击工具栏中的颜色圆圈快速切换画笔颜色：

- 🔴 红色
- 🔵 蓝色
- 🟢 绿色
- 🟡 黄色
- 🟠 橙色
- 🟣 紫色
- 🩷 粉色
- ⚫ 黑色

### 笔粗切换

点击工具栏中的圆点快速切换笔粗：

- ⚫ 细（2px）
- ⚫ 中（4px）
- ⚫ 粗（6px）
- ⚫ 特粗（8px）

### 橡皮大小切换

点击工具栏中的圆点快速切换橡皮大小：

- ⚫ 小（10px）
- ⚫ 中（20px）
- ⚫ 大（30px）

## 快捷键提示

::: tip 提示
在设置中启用"显示工具提示"选项，鼠标悬停在按钮上时会显示功能说明和快捷键。
:::

## 自定义快捷键

当前版本不支持自定义快捷键，但我们计划在未来版本中添加此功能。

## 快捷键冲突

如果快捷键与其他应用冲突，可以：

1. 关闭冲突的应用
2. 修改其他应用的快捷键设置
3. 等待未来版本支持自定义快捷键

## 常用操作流程

### 快速批注流程

1. `Ctrl+O` 打开 PDF
2. 点击画笔工具
3. 选择颜色和粗细
4. 左键拖动绘制批注
5. `Ctrl+S` 保存批注

### 快速修改流程

1. 点击橡皮工具
2. 选择橡皮大小
3. 在错误批注上拖动擦除
4. 点击画笔工具继续批注
5. `Ctrl+S` 保存批注

### 快速浏览流程

1. `Ctrl+鼠标滚轮` 调整缩放
2. `鼠标滚轮` 上下滚动
3. `Shift+鼠标滚轮` 左右滚动

## 效率技巧

### 技巧 1：快速切换工具

使用鼠标点击工具栏按钮比使用快捷键更快：

- 画笔和橡皮工具只需一次点击
- 颜色和粗细选择也只需一次点击

### 技巧 2：缩放技巧

- 按住 `Ctrl` 键，使用鼠标滚轮快速调整缩放
- 缩放时页面会自动居中，无需手动调整位置

### 技巧 3：拖拽打开

- 直接从文件资源管理器拖拽 PDF 文件到窗口
- 比使用 `Ctrl+O` 更快捷

### 技巧 4：最近文件

- 在主页点击最近文件卡片快速打开
- 比每次浏览文件夹更高效

### 技巧 5：自动保存

- 在设置中启用"自动保存批注"
- 无需记住按 `Ctrl+S`，关闭时自动保存
